# Contributing a Jupyter Notebook

## Submissions

To submit your [Jupyter notebook](http://jupyter.org/) as a candidate for inclusion in the [QuantEcon Notebook Gallery](https://github.com/QuantEcon/notebook-gallery), please submit a pull request to the [notebook-gallery repository](https://github.com/QuantEcon/notebook-gallery).

You can also reach out via [contact@quantecon.org](mailto:contact@quantecon.org) if you have questions.

## Style Guide

When contributing a notebook, please ensure:

1. The notebook runs from top to bottom without errors
2. All dependencies are clearly documented
3. Include a brief summary of what the notebook demonstrates
4. Follow QuantEcon coding conventions where applicable
